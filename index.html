• Produtos até 50 USD são tributados em 17%* de ICMS

* Pode variar de acordo com cada estado

• Produtos acima de 50 USD são tributados em 60% + ICMS

Cálculo do ICMS:

Base Cálculo = (Valor Aduaneiro + Imposto de Importação(60%) / ( 1 - Alíquota ICMS )

Valor de ICMS Devido = Base Cálculo * Alíquota do ICMS
